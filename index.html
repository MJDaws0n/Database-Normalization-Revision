Database Normalistaion

Deffinition

Database normalisation is the process of organising data into related tables to reduce duplication and improve integrity.

Example

If TeacherRoom depends on TeacherName instead of StudentID, move it to a separate table.

First Normal Form (1NF)

No repeating records

Each records must be unique and must not be identical to any others.

Each field contains only one value

A field in the table must only contain one value per record and no more.

No repeating columns

Each column cannot be a duplicate of the late. Such as: subject1, subject2, subject3

Seconds Normal Form (2NF)

Must already be in 1NF

Must already follow all the rules of First Normal Form

Each field contains only one value

Basically no values
